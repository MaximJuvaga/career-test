# Карьера: Твой путь 
Веб-приложение для автоматизации процесса профориентации, анализа профессиональных склонностей учащихся и формирования персонализированных рекомендаций по направлениям подготовки и профессиям. Разработано в рамках выпускной квалификационной работы для Тульского государственного университета (ТулГУ).

##  О проекте

Система предназначена для использования в образовательных учреждениях (школы, колледжи, вузы) и центрах карьеры. Она помогает абитуриентам сделать осознанный выбор будущей профессии на основе тестирования, а преподавателям — анализировать групповые тенденции.

Основные цели:
Автоматизация анализа профессиональных склонностей.
Предоставление рекомендаций по направлениям обучения ТулГУ.
Интеграция с рынком труда (вакансии hh.ru).
Визуализация результатов тестирования.

##  Возможности

## Для абитуриента
Интерактивный профориентационный тест с вопросами по различным категориям склонностей.
Круговые диаграммы выраженности профессиональных интересов (Chart.js).
Список подходящих направлений подготовки и профессий.
Подборка актуальных вакансий по рекомендованным профессиям (через API hh.ru).
Сохранение истории прохождения тестов в личном кабинете.

### Для преподавателя
Просмотр сводной статистики по всем абитуриентам.
Поиск результатов по дате, институту и логину пользователя.
Возможность анализа групповых тенденций для профориентационной работы.
